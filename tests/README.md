# OCRテスト手順

## テスト用サンプルファイル

| ファイル | 内容 |
|----------|------|
| `sample-ocr.png` | OCRテスト用画像（800×400px、日本語テキスト含む） |

期待されるOCR結果：
```
これはOCRテスト用のサンプルテキストです。
読書メモのテスト。
```

---

## テスト手順

### 前提条件
- Gemini APIキーが設定済みであること（設定画面から登録）

### 手順

1. ブラウザで以下のURLを開く
   https://bluebird4320.github.io/book-manager/

2. 下部ナビの「**登録**」をタップ

3. 「**画像OCR**」タブを選択

4. 「**端末から画像・PDFを選択**」エリアをタップ

5. `tests/sample-ocr.png` を選択してアップロード

6. スピナーが表示され、Gemini APIで解析が始まる

7. OCR結果テキストエリアに以下が表示されれば **OK**：
   ```
   これはOCRテスト用のサンプルテキストです。
   読書メモのテスト。
   ```

8. タイトル欄に書名を入力して「保存する」
